# audiora 

## Brief
Make LoRa radio modulation audible.
Listen radiofrequencies until signal detection > start recording > process the signal to make it audble > output the signal
It uses the RTL-SDR Blog v4

## Quick start
- To install the RTL-SDR driver go here -> https://www.rtl-sdr.com/rtl-sdr-quick-start-guide/
- To run the project: "python audiora.py"
